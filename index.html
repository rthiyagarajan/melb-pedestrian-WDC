Melb-pedestrian-wdc

Tableau Wed Data Connector for Melbourne city pedestrian counts

View on GitHub Download .zip Download .tar.gz

This is a Tableau 10 beta v2 Web Data Connector to access the Melbourne City dataset of Pedestrian Counts based on sensors placed across the city.

The dataset is currently limited to data from 2016 via the SODA API query. IF required data is available back to 2009.

Explore the dataset here:

  • https://data.melbourne.vic.gov.au/Transport-Movement/Pedestrian-Counts/b2ak-trbp#column-menu

Find the API here:

  • https://dev.socrata.com/foundry/data.melbourne.vic.gov.au/mxb8-wn4w
